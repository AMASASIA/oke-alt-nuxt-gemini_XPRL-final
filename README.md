# OKE-ALT Nuxt Gemini (static demo)

This repository reproduces the page at https://oke-alt-nuxt-gemini.vercel.app/

## Features
- Minimal Nuxt 3 (ssr: false) single-page demo
- Wallet status placeholder (GemWallet connect button)
- Simple "Generate NFT" UI and preview area

## How to run locally
1. Install Node.js (18+ recommended).
2. In repository folder, run:
   ```
   npm install
   npm run dev
   ```
3. Open http://localhost:3000

## Notes
- This is a front-end demo skeleton. Backend integrations (Gemini AI, XRPL) are placeholders and must be implemented separately.
